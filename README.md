# Content Intelligence OS

A full-stack content strategy dashboard built with React, Vite, and Tailwind CSS.

## Features

- **Dashboard** — Overview of key metrics and recent activity
- **Analytics** — Track post performance across platforms
- **Ideas Hub** — Manage and organize content ideas
- **Content Board** — Kanban-style content pipeline
- **Insight Engine** — AI-generated insights from your analytics data
- **Trend Radar** — Analyze trending topics and competitors
- **Idea Loop** — Generate new content ideas from insights and trends

## Tech Stack

- React 18 + Vite
- Tailwind CSS v3
- Zustand (state management)
- Recharts (data visualization)
- @hello-pangea/dnd (drag and drop)

## Theme

White & orange design system — clean, minimal, and professional.

## Getting Started

```bash
npm install
npm run dev
```
